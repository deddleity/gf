Optimal Child Theme
http://www.studiopress.com/themes/optimal

INSTALL
1. Upload the Optimal child theme folder via FTP to your wp-content/themes/ directory. (the Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Optimal theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking.

WIDGETS

Primary Sidebar - This is the primary sidebar if you are using the Content/Sidebar, Sidebar/Content, Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.

Secondary Sidebar - This is the secondary sidebar if you are using the Content/Sidebar/Sidebar, Sidebar/Sidebar/Content or Sidebar/Content/Sidebar Site Layout option.

Slider - This is the slider widget section of the homepage.

Welcome - This is the welcome section of the homepage.

Home Feature Sidebar - This is the sidebar of the homepage middle section.
Home Feature #1 - This is the first column of the homepage middle section.
Home Feature #2 - This is the second column of the homepage middle section.

Home Bottom Sidebar - This is the left sidebar near the bottom of the homepage.
Home Featured Posts - This is the featured posts section were you can place the Genesis Featured Posts widget to display your recent posts.

Home Bottom Message - This is the bottom section of the homepage were you can place text, quotes, twitter updates, or whatever you like.

SLIDESHOW
The slideshow that is underneath the header is using the Genesis Responsive Slider plugin which can be downloaded here - http://wordpress.org/extend/plugins/genesis-responsive-slider. Upload the Genesis Slider folder via FTP to your wp-content/plugins/ directory and then activate it. You will also need to configure the plugin by going to the Plugins > Genesis Slider page in your dashboard.
